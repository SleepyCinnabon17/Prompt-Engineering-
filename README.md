📚 About This Repository

This project is a deep exploration of prompt engineering, combining:

🪞 Mirror Journal (Part A) → Using AI to challenge a real personal belief
⚙️ Technical Practicum (Part B) → Systematic experimentation with prompting strategies

Instead of treating LLMs like “smart tools,” this project treats them as:

⚠️ Systems that appear intelligent but require careful control, skepticism, and evaluation.

🪞 Mirror Journal – Core Idea

A 7-session structured experiment where a single belief is interrogated using progressively advanced prompting techniques.

**🧠 Belief Explored**

“All religions are misogynistic at their core…”

Not exactly light dinner conversation, but sure, go big or go home.

**🔍 Mirror Sessions Breakdown**
Session	Technique	What Happened
1	Zero-shot	AI gave broad, diplomatic answer → missed specificity
2	Role Prompting	Skeptic vs Defender → both converged to power structures
3	Few-shot	Personal examples improved nuance but echoed bias
4	Chain-of-Thought	Forced structured reasoning → exposed logical gaps
5	Self-Consistency	Multiple runs revealed stability ≠ truth
6	Metacognitive	AI identified uncertainty but missed key blind spots
7	Synthesis	Final belief refined, not replaced
⚠️ Key Insight from Mirror Journal

The AI doesn’t argue with you.
It negotiates with your prompt.

Even when challenged, it tends to:

Default to safe, balanced conclusions
Avoid committing strongly
Mirror your framing unless tightly controlled

**📚 RAG (Retrieval-Augmented Generation)**
_Context grounding improved factual quality_
But:
Garbage context = confident garbage output

🧬 Core Takeaway**
LLMs don’t think.
They simulate thinking.

And if you’re not careful:

They’ll simulate correctness too.

The most powerful technique wasn’t CoT or RAG
It was forcing the model into constraints it couldn’t escape

Because:

_**The moment you remove ambiguity, the illusion starts breaking.
